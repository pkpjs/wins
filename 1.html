CVE-2024-27956 - wp-automatic SQL Injection 취약점

🔍 취약점 개요

  • CVE 번호: CVE-2024-27956
  • 플러그인: wp-automatic
  • 취약점 유형: SQL Injection
  • 설명: 외부 입력값을 적절히 검증하지 않아 SQL Injection이 발생할 수 있습니다.

🛠️ 대응 방안

  • 플러그인을 최신 버전으로 업데이트합니다.
  • 웹 애플리케이션 방화벽(WAF)을 사용하여 악의적인 요청을 차단합니다.
  • 취약점이 발생한 입력값에 대한 검증 로직을 강화합니다.
